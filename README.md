# Scala 3 Benchmark Data

This repository contains data generated by [lampepfl/scala3-benchmarks](https://github.com/lampepfl/scala3-benchmarks).

See the ["Results structure"](https://github.com/lampepfl/scala3-benchmarks?tab=readme-ov-file#results-structure) section in that repository’s README.

## Checking warmup health

*(Entirely written by Claude Fable 5)*

`python3 checkWarmups.py [-v] [filter]` scans the raw per-iteration data and
reports, for every benchmark, whether its JMH warmup is sufficient.
